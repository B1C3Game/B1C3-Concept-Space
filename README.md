# Concept Space Visualization
A 3D visualization app for exploring relationships between abstract concepts, built with React and Plotly.

## Main Features

- Visualizes a set of concepts in a 3D space using Plotly, with axes for Standardization, Intelligence Required, and Cognitive Distance
- Shows relationships and shifts between concepts with red dashed lines, highlighting communication improvements
- Displays a convex hull (valid region) for all concepts
- Interactive plot with tooltips and responsive controls
- Explanatory panel describing the meaning of each axis and the significance of concept shifts
