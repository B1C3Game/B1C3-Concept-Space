# Concept Space Visualization

A 3D visualization app for exploring relationships between abstract concepts, built with React and Plotly.

## Live Demo

[View Demo](#) <!-- Add link after Pages is up -->

## Main Features

- 3D interactive plot with axes for Standardization, Intelligence Required, and Cognitive Distance
- Shift arrows (red dashed lines) showing communication improvements between concepts
- Convex hull showing the valid region for all concepts
- Tooltips and responsive controls
- Explanatory panel describing each axis and the significance of concept shifts

## Future Features

- Add new concepts dynamically
- Improved navigation and concept management
- Enhanced UX for a more intuitive experience

## Run Locally

```bash
npm install
npm run dev
```
